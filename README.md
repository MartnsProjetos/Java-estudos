# Estudos em Java

Este repositório reúne meus estudos, exercícios e projetos desenvolvidos durante minha jornada de aprendizado na linguagem Java.

## Estrutura

- **/fundamentos**  
  Conceitos básicos de Java: sintaxe, variáveis, operadores, entrada e saída.

- **/estruturas-controle**  
  Condicionais (`if`, `else`, `switch`) e laços (`for`, `while`, `do-while`).

- **/poo**  
  Programação Orientada a Objetos: classes, objetos, herança, encapsulamento, polimorfismo.

- **/colecoes**  
  Arrays, listas (`ArrayList`), conjuntos (`HashSet`) e mapas (`HashMap`).

- **/desafios**  
  Exercícios práticos e problemas de lógica para treinar.

- **/projetos**  
  Mini projetos Java para aplicar os conhecimentos aprendidos.

## Tecnologias

- Java (JDK 17+)
- IDE: IntelliJ IDEA / VS Code
- Git & GitHub

## Objetivo

Aprender Java com base na prática, reforçar lógica de programação e montar um portfólio para oportunidades como desenvolvedor júnior, especialmente com foco no setor bancário.
